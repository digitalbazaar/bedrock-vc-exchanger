## Install

- Node.js 14+ is required.

### NPM

To install via NPM:

```
npm install --save @bedrock/vc-exchanger
```

### Development

To install locally (for development):

```
git clone https://github.com/digitalbazaar/bedrock-vc-exchanger.git
cd bedrock-vc-exchanger
npm install
```
